# Кодекс поведения

## Наши стандарты

Мы стремимся к уважительному и конструктивному общению:

- уважение к участникам и их времени;
- конкретная обратная связь по коду и идеям;
- допустимы разногласия по техническим решениям без личных нападок.

Недопустимо:

- оскорбления, дискриминация, травля;
- публикация чужих личных данных без согласия;
- иное поведение, которое делает участие в проекте некомфортным.

## Сообщить о нарушении

Напишите автору проекта через GitVerse: [Max_Cherep](https://gitverse.ru/Max_Cherep).

Опишите ситуацию и ссылки на сообщения/issue. Ответ — в разумные сроки.

## Меры

При нарушении кодекса автор может закрыть issue/PR, заблокировать участника или принять иные меры по усмотрению.
